# Le Démocrate de la Broye

Journal suisse, successeur de la *Feuille d'avis de Payerne*.
Paraît pour la première fois le samedi 18 avril 1846.

Pour plus d'informations, consulter : Blaser Fritz, *Bibliographie de la presse suisse*, vol. 1/2, Basel, Birkhäuser, 1956 (Quellen zur Schweizer Geschichte).
